# 🤝 Negotiation Styles

| Region | Style | Notes |
|--------|-------|-------|
| USA | Direct, fast | Get to the point |
| UK | Reserved, polite | Understated approach |
| Germany | Direct, logical | Facts over feelings |
| France | Intellectual, debate | Enjoy discussion |
| Japan | Indirect, consensus | "Nemawashi" process |
| China | Relationship-based | "Guanxi" important |
| Korea | Relationship + hierarchy | Age/status matters |
| India | Flexible, relationship | "Jugaad" (creative solutions) |
| Middle East | Relationship first | Trust before business |
| Brazil | Personal, warm | Build friendship first |
| Mexico | Personal, warm | Trust building essential |
| Russia | Direct, tough | Strong negotiation expected |
| Australia | Direct, casual | Informal approach |
| Africa (varies) | Relationship-based | Trust building |

### Decision Making
| Region | Speed | Process |
|--------|-------|---------|
| USA | Fast | Individual decision |
| Germany | Moderate | Logical analysis |
| Japan | Slow | Consensus building |
| China | Slow | Hierarchical approval |
| Korea | Moderate | Senior approval |
| Middle East | Slow | Trust-based |

|----------|--------|---------|

| Chinese | "Zài shāngliang" | "Let's discuss more" (maybe) |
| Korean | "Goryeol hagetsseumnida" | "I'll consider it" (soft no) |
| Arabic | "Inshallah" | "God willing" (maybe) |
| Hindi | "Sochke batata hoon" | "I'll think about it" (maybe) |

## Do's and Don'ts

| Do | Don't |
|----|-------|
| Build relationships (Asia, ME) | Rush to close deal (Asia) |
| Be patient (Japan, China) | Push for immediate decision |
| Respect hierarchy (Asia) | Go over someone's head |
| Bring gifts (China, Japan) | Give clocks (China) |
| Use right hand (Muslim) | Use left hand (Muslim) |
| Dress formally (Asia) | Dress too casual |
| Learn basic greetings | Be culturally ignorant |

## Safe Practices

- **Research partner's culture**
- **Build relationship first** (most non-Western)
- **Be patient** with decision-making
- **Respect hierarchy**
- **When in doubt**, ask local advisor
