# 🥢 Chopstick Etiquette Across Asia

| Taboo | Region | Meaning | Offense Level | Notes |
|-------|--------|---------|---------------|-------|
| Sticking upright in rice | China, Japan, Korea, Vietnam | Funeral rite / death | Extreme | Never do this |
| Passing food chopstick-to-chopstick | Japan | Cremation ceremony | High | Use serving chopsticks |
| Crossing chopsticks | China, Japan | Death symbol | High | Avoid |
| Pointing with chopsticks | Japan, China, Korea | Rude | Medium | Use open hand |
| Drumming on bowls | Japan, China, Korea | Begging | Medium | Considered childish |
| Sucking on chopsticks | Japan, China, Korea | Rude | Medium | Place on rest |
| Using one chopstick | All | Incomplete / rude | Low | Always use pair |
| Stabbing food | All | Barbaric | Medium | Use chopsticks to lift |
| Hovering over dishes | All | Indecisive / rude | Low | Choose quickly |

## Region-Specific Rules

### China
- Place chopsticks on rest or bowl when not eating
- Tap bowl rim with chopsticks is acceptable
- Elders eat first
- Use serving chopsticks when available

### Japan
- Use provided chopstick rest
- Never pass food directly (use serving utensils)
- Say "itadakimasu" before eating
- Say "gochisousama" after eating

### Korea
- Metal chopsticks are standard
- Hold rice bowl close to mouth
- Elders eat first
- Use spoon for rice and soup

### Vietnam
- Similar to Chinese etiquette
- Chopsticks often shorter
- Use soup spoon for broths
- Respect elders

### Taiwan
- Similar to Chinese etiquette
- Use serving chopsticks
- Respect elders
- Place on rest when not eating

### Hong Kong
- Similar to Chinese etiquette
- Use serving chopsticks
- Respect elders
- Place on rest when not eating

### Singapore
- Similar to Chinese etiquette
- Use serving chopsticks
- Respect elders
- Place on rest when not eating

### Malaysia
- Similar to Chinese etiquette
- Use serving chopsticks
- Respect elders
- Place on rest when not eating

### Philippines
- Chopsticks not traditional
- Fork and spoon more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Indonesia
- Chopsticks not traditional
- Hands more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Thailand
- Chopsticks for noodle soups only
- Fork and spoon for rice dishes
- Respect elders
- Place on rest when not eating

### Myanmar
- Chopsticks not traditional
- Hands more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Cambodia
- Chopsticks not traditional
- Hands more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Laos
- Chopsticks not traditional
- Hands more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Mongolia
- Chopsticks not traditional
- Hands and knife more common
- If using chopsticks, follow Chinese rules
- Respect elders

### Brunei
- Similar to Malay etiquette
- Use serving chopsticks
- Respect elders
- Place on rest when not eating

## Chopstick Materials

| Material | Region | Notes |
|----------|--------|-------|
| Bamboo | China, Vietnam | Most common |
| Wood | Japan, Korea | Traditional |
| Metal | Korea | Standard |
| Plastic | Japan, modern | Disposable |
| Ivory | Japan, formal | Expensive |
| Lacquered | Japan, formal | Elegant |

## Safe Practices

- Place chopsticks on provided rest when not in use
- Use serving utensils when available
- Follow the lead of your hosts
- When unsure, ask politely
