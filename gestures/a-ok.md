# 👌 A-OK Sign

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | Everything is fine / OK | None | Very common |
| France | Zero / worthless | Low | Can be dismissive |
| Brazil | Highly offensive (equivalent to middle finger) | Extreme | Absolutely avoid |
| Turkey | Offensive (insult) | High | Avoid |
| Germany | Asshole (Arschloch) | High | Insulting |
| Japan | Money / wealth | None | Business context |
| China | Zero / nothing | None | Used in counting |
| Italy | Worthless / zero | Medium | Context-dependent |
| Middle East | Offensive sexual gesture | High | Avoid |
| Australia | OK / good | None | Common |

## Safe Alternatives

- 👍 Thumbs up (avoid in Middle East, West Africa)
- Nodding head
- Verbal "OK", "fine", "good"
- 🤝 Handshake
