# 🤙 Beckoning Gesture (Come Here)

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | Come here / approach | None | Common gesture |
| Japan | Come here (palm down, fingers curling) | None | Traditional way |
| Philippines | Very offensive (used only for dogs) | Extreme | Never use toward people |
| China | Come here (palm down) | None | Normal usage |
| Europe | Come here | None | Generally understood |
| South Korea | Come here | None | Similar to Japan |
| Indonesia | Impolite to use toward superiors | Medium | Use with caution |
| Singapore | Rude to use toward elders | Medium | Respect age hierarchy |

## Safe Alternatives

- Palm up, fingers beckoning (more polite in many Asian cultures)
- Verbal "come here" or "follow me"
- Waving hand toward yourself
- Head nod in direction you want person to go
