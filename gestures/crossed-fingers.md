# 🤞 Crossed Fingers

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | Good luck / hoping for something | None | Very common |
| Vietnam | Female genitalia / vulgar | Extreme | Highly offensive |
| China | Number 10 (crossed fingers) | None | Used in counting |
| Europe (general) | Good luck | None | Widely understood |
| Australia | Good luck | None | Common |
| South Korea | Good luck | None | Western influence |

## Safe Alternatives

- 🍀 Four-leaf clover reference (verbal)
- Verbal "good luck" / "fingers crossed"
- 👍 Thumbs up
- 🙏 Prayer hands
