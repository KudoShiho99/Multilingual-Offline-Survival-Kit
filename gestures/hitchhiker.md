# 👍 Hitchhiker Thumb

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | I need a ride / hitchhiking | None | Common gesture |
| Greece | Highly offensive (go to hell) | Extreme | Historical insult |
| Middle East | Offensive gesture | High | Avoid |
| South Africa | Offensive in some contexts | Medium | Context-dependent |
| West Africa | Can be offensive | Medium | Use with caution |
| Europe (general) | Hitchhiking request | None | Generally understood |
| Australia | Hitchhiking | None | Common |

## Safe Alternatives

- 👋 Wave to flag down vehicles
- Raise hand with palm facing traffic
- Verbal request for directions or help
- Use official transportation services
