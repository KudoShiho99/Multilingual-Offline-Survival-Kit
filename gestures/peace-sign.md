# ✌️ Peace Sign / V Sign

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | Peace, victory | None | Very common, especially at events |
| UK/Australia (palm inward) | Extremely offensive (equivalent to middle finger) | Extreme | Always show palm outward |
| Japan | Victory, peace | None | Popular in photos |
| Germany | Victory (palm outward) | None | Historical WWII context |
| China | Two / number 2 | None | Used when counting |
| South Korea | Victory / peace | None | Common in photos |
| Ireland | Peace | None | Widely understood |
| Europe (general) | Peace, victory | None | Palm outward is safe |

## Safe Alternatives

- 👍 Thumbs up (avoid in Middle East)
- 👌 OK sign (avoid in Brazil, Turkey)
- Peace (verbal)
- 🙏 Hands together (respectful in many cultures)
