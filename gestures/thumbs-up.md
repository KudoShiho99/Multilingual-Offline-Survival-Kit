# 👍 Thumbs Up

| Region | Meaning | Offense Level | Notes |
|--------|---------|---------------|-------|
| USA/UK | Good, approval, agreement | None | Very common |
| Middle East (Iran, Iraq) | Extremely offensive (equivalent to middle finger) | Extreme | Avoid completely |
| West Africa | Aggressive gesture, similar to middle finger | High | Avoid |
| Germany | One (counting) | None | Used when counting |
| Japan | Father / boss | None | Informal |
| Australia | Same as middle finger in some contexts | Medium | Context-dependent |
| China | Good / good job | None | Very positive |
| Europe (general) | Approval, positive | None | Widely understood |

## Safe Alternatives

- 👌 OK sign (but avoid in Brazil, Turkey, Germany)
- Nodding head
- Verbal "good", "well done", "agreed"
- 👋 Wave for greeting
