# 🙇 Bowing Customs

| Region | Custom | Offense Level | Notes |
|--------|--------|---------------|-------|
| Japan | Bow angle = respect level | Low (wrong angle) | 15° casual, 30° polite, 45° deep |
| Korea | Slight bow with handshake | Low | Less formal than Japan |
| China | Slight nod/bow | None | Less common now |
| Thailand | Wai (prayer hands + bow) | Low | Height of hands = respect |
| Vietnam | Slight bow | None | Similar to China |
| India | Namaste (prayers + bow) | None | Common greeting |
| Cambodia | Sampeah (prayer hands) | None | Similar to Thailand |
| Laos | Similar to Thailand | None | Slight bow |
| Myanmar | Joint palms at chest | None | Respectful |
| Mongolia | Slight bow, may offer hatag | None | Formal occasions |

## Bow Angles (Japan)

| Angle | Use Case | When |
|-------|----------|------|
| 15° | Casual greeting | Friends, colleagues |
| 30° | Polite greeting | Business, strangers |
| 45° | Deep respect | Apology, gratitude, superiors |
| 90° | Deepest respect | Formal ceremonies |

## Regional Variations

### Japan
- Most formal bowing culture
- Longer bow = more respect
- Keep eyes lowered
- Don't make eye contact while bowing

### Korea
- Lighter bow than Japan
- Often combined with handshake
- Elders receive deeper bow
- Business: slight bow OK

### Thailand (Wai)
- Palms together at chest
- Higher hands = more respect
- Return wai to children casually
- Don't wai to service staff

### India (Namaste)
- Palms together at chest
- Slight bow with it
- "Namaste" verbal greeting
- Common in spiritual context

### Cambodia (Sampeah)
- Similar to Thai wai
- Height of hands varies
- Respect for elders
- Formal occasions

## Safe Practices

- **Observe local behavior** first
- **Slight bow** is safe default
- **Don't over-bow** in non-bowing cultures
- **Return greeting** appropriately
- **When in doubt**, nod respectfully
