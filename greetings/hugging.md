| Region | Custom | Offense Level | Notes |
|--------|--------|---------------|-------|
| USA/Canada | Common among friends | Low (strangers) | Hug greeting |
| Latin America | Very common | None | Warm, personal |
| Brazil | Hug + kiss on cheek | None | Very friendly |
| Mexico | Abrazo (hug) common | None | Warm greeting |
| Argentina | Hug common | None | Friendly |
| France | Double cheek kiss preferred | Low (hug) | More formal |
| Italy | Hug among close friends | None | Warm |
| Spain | Hug common | None | Friendly |
| Germany | Rare, handshake preferred | Medium | Formal |
| UK | Rare, awkward | Medium | Reserved culture |
| Japan | No physical contact | High | Bow instead |
| China | No hugging | High | Handshake/nod |
| Korea | No hugging | High | Bow/handshake |
| India | Namaste preferred | Medium | Conservative |
| Middle East | Same gender: OK; Opposite: no | High (opposite gender) | Conservative |
| Australia | Hug among friends | Low | Casual |
| Russia | Hug among close friends | Low | Not with strangers |
| Africa (varies) | Varies by region | Medium | Observe locals |
| Thailand | No hugging (wai instead) | Medium | Conservative |
| Turkey | Same gender: OK; Opposite: no | High (opposite gender) | Conservative |

## Key Rules

1. **Same gender hugging** generally OK in most cultures
2. **Opposite gender hugging** avoid in conservative areas
3. **Observe locals** before initiating
4. **Let them initiate** if unsure
5. **Brief hug** is safer than long hug

## Hugging Styles

| Style | Region | Description |
|-------|--------|-------------|
| One-armed | USA, UK | Brief, casual |
| Two-armed | Latin America | Warm, full body |
| Side hug | Conservative | Less intimate |
| Back pat | USA, men | Shows camaraderie |
| Squeeze | Latin America | Enthusiastic |

## When to Avoid

- First meeting with stranger
- Conservative religious areas
- Professional/business settings
- Opposite gender in conservative cultures
- During pandemic/illness

## Safe Practices

- **Observe locals** first
- **Handshake is safer** default
- **Let them initiate** hug
- **When in doubt**, smile and nod
- **Respect personal space**
